# Juice
Juice programming language

## About:
- A simple programming language built using clang inspired by rustlang!

## Get Compiler:
- get juicec by running this command
```
$ cd juicesrc/
$ make
$ mv juice /usr/local/bin
$ juice
```

## Tutorial:
- basic print function
```ruby
println!(`Hello World`)
```
